------------ METHOD 1: SIMPLEST WAY (Recommended) ------------

1. Create a folder on your computer (e.g., "Portfolio" or "myPortfolio")
2. Put ALL files in that folder
3. Double-click "index.html"
4. Your portfolio opens in your browser
5. DONE! ✅

------------ METHOD 2: USING LOCAL SERVER (Windows) -----------

Windows 10/11:
1. Download and install Python from python.org
2. Open Command Prompt
3. Go to your portfolio folder: cd "C:\path\to\your\portfolio"
4. Run: python -m http.server 8000
5. Open browser to: http://localhost:8000
6. Press Ctrl+C to stop the server

------------ METHOD 3: USING LOCAL SERVER (Mac/Linux) --------

1. Open Terminal
2. Go to your portfolio folder: cd ~/Desktop/portfolio
3. Run: python3 -m http.server 8000
4. Open browser to: http://localhost:8000
5. Press Ctrl+C to stop the server

------------ TROUBLESHOOTING -----------

❌ Problem: Blank page or won't load
✓ Solution: Check all files are in same folder
✓ Solution: Try opening with different browser
✓ Solution: Check internet (for loading external libraries)

❌ Problem: Images showing as broken
✓ Solution: Make sure image files are in same folder
✓ Solution: Check spelling: myProfile.JPG, c++.png, html.png, css.png, cs.png
✓ Solution: For Mac users, check if file extensions are correct

❌ Problem: Styles look weird/broken
✓ Solution: Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
✓ Solution: Clear browser cache and reload
✓ Solution: Try a different browser

❌ Problem: Theme toggle not working
✓ Solution: Refresh the page
✓ Solution: Try incognito/private browsing mode
✓ Solution: Check browser allows localStorage

❌ Problem: Hamburger menu not working on mobile
✓ Solution: Make sure script.js is in same folder
✓ Solution: Check browser console for errors (F12)

------------ COMMON FILE ISSUES -----------

Issue: File named "c++.png" won't load
Solution: Windows may have issues with "++" in filename
Fix: Use this instead in HTML: "c__.png" (two underscores)

Issue: Mac doesn't show .JPG files properly
Solution: Try uppercase: myProfile.JPG
Solution: Or convert to .jpg (lowercase)
